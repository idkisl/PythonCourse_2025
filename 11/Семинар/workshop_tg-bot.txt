Семинар 11
Создание телеграмм-бота

1. Установите -> pip install python-telegram-bot
2. Получите API KEY у BotFather (@BotFather)
3. В PyCharm создайте файл echo-bot.py
    -> Напишите код эхо-бота, запустите.
    -> Проверьте работоспособность кода
4. В PyCharm создайте файл hello-bot.py
    -> Напишите бот, который здоровается с человеком, отправившем ему сообщение.
    -> Проверьте работоспособность кода
5. В PyCharm создайте файл easy-math-bot.py
    -> Напишите бот, который умеет решать квадратное уравнение. ( на вход подаются три числа - коэфициенты a, b и c квадратного трехчлена)
    -> Напишите команду /help - описывающую умения Вашего бота.
    -> Напишите команду /start - здоровающуюся с пользователем и выводящую на экран краткую информацию о возможностях использования бота.
6. В PyCharm создайте файл currency-converter-bot.py
    -> Напишите бот, который умеет переводить одну валюту в другую.
    -> Бот сначала спрашивает из какой валюты Вы хотите перевести сумму. Затем в какую. После этого вводится сумма, которую необходимо перевести. 
    -> Напишите команды /help, /start
    -> Для выбора валюты реализуйте ReplyKeyboardMarkup
    -> Воспользуйтесь API из предыдущих занятий для получения точного перевода.
7. В PyCharm создайте файл electronic-diary-bot.py
    -> Напишите бот, который иммитирует работу электронного журнала
    -> В словаре (или json-файле) хранится информация о преподавателях, зарегистрированных в системе.
    -> При вызове команды /start необходимо уточнить, пользователь является преподавателем или учеником. Если учеником, ему нужно предложить выбрать преподавателя из списка зарегистрированных преподавателей.
    -> Если при вызове /start пользователь выбрал роль преподавателя, ему необходимо представиться, данные записываются в импровизированную базу данных.
    -> Каждый преподаватель может выставить оценку одному из своих зарегистрированных студентов.
    -> Каждый студент может увидеть список своих оценок.
    -> Для упрощения системы, каждый студент может учиться лишь у одного преподавателя по одному предмету.
